# Member Feature Progress Tracking

Tracking the implementation status of all member-facing features for the CredTrust Cooperative Society.

## 1. Member & Share Management
- [x] **Member Profile UI**: High-fidelity personal details and address blocks.
- [x] **KYC Verification**: Multi-step identity and address proof workflow.
- [x] **Share Capital Interface**: Viewing share holdings and certificate.
- [ ] **Purchase Shares**: Functional flow to increment share capital.
- [ ] **Membership Ledger**: Historical view of share contributions and exits.
- [ ] **Dividend Distribution**: View allocated dividends from annual profits.

## 2. Deposit & Thrift
- [x] **Relationship Overview**: Summary of account balances and savings cards.
- [x] **FD/RD Application**: Dedicated forms for Fixed and Recurring Deposits.
- [x] **Online Collections**: Integration for monthly RD installments.
- [ ] **Maturity Manager**: Payout and renewal workflows for matured deposits.

## 3. Loan Management
- [x] **Loan Selection Landing**: Visual menu of various loan products.
- [x] **Detailed Loan Forms**: Specialized application flows for Gold, SHG, Emergency loans.
- [x] **Digital Repayment Hub**: Unified portal for Loan EMI payments.
- [ ] **Repayment Tracker**: EMI calendar and past repayment history.
- [ ] **Loan Calculator**: Interest and tenure simulation tools.
- [ ] **Collateral Management**: Uploading and tracking loan security documents.

## 5. Profile & Identity
- [x] **Member Profile**: Comprehensive summary of member status and details.
- [x] **KYC Vault**: Visual dashboard for identity document status.
- [x] **Nominee Management**: Specialized interface for beneficiary assignment.
- [ ] **Account Settings**: Advanced security and preference controls.

## 4. Digital Banking Services
- [ ] **Transaction History**: Searchable and filterable account statements.
- [ ] **Spend Analysis**: Visual categorization of member expenditures.
- [ ] **Service Requests**: Online requests for passbooks, cheque books, and certificates.
